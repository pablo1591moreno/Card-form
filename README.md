# Card form

Formulario de tarjeta de credito
reprica los datos en la tarjeta de ejemplo 
valida los ingresos de datos 
confirma ingreso si todos los datos estan OK

## Recursos utilizados

- HTML
- SASS
- JAVASCRIPT
- BOOTSTRAP
